# JNTETRI

## Build Instructions

### Windows

Open `JNTETRI.sln` with Visual Studio and press F5.

#### MSYS2:

```
pacman -S mingw-w64-x86_64-SDL2
```

### Linux

You'll need scons, boost and [JNGL](https://github.com/jhasse/jngl) installed. Type

```
scons -h
```

to get a list of build options available.
